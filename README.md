> npm i #Intalador de paquetes.

_Ejemplo Yarn_

> cd front-react.
> yarn #Intalador de paquetes.
> yarn start #Levanta app en puerto 3001

_Para el BACK-END_
_Ejemplo Npm_

> cd back-express.
> npm i #Intalador de paquetes.
> npm run dev #Levanta app en puerto 3000

_Ejemplo Yarn_

> cd back-express.
> yarn #Intalador de paquetes.
> yarn start:dev #Levanta app en puerto 3000
